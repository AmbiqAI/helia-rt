## Getting Started

1. Fork the repository and create a feature branch.
2. Read the [Architecture](architecture.md) page to understand the source layout.
3. Make your changes, ensuring tests pass locally.
4. Open a pull request against `main`.

## Code of Conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/). Be respectful and constructive.
